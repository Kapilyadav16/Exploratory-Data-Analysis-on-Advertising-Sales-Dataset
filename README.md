# README.md

## Exploratory Data Analysis on Advertising Sales Dataset

### Project Overview

This project performs Exploratory Data Analysis (EDA) on the Advertising Sales dataset to identify relationships between advertising channels and sales performance.

### Objectives

* Understand the structure of the dataset
* Perform data cleaning and preprocessing
* Analyze statistical summaries
* Visualize data distributions and relationships
* Identify factors influencing sales
* Generate actionable business insights

### Technologies Used

* Python
* Jupyter Notebook
* Pandas
* NumPy
* Matplotlib
* Seaborn

### Installation

Clone the repository:

```bash
git clone https://github.com/your-username/EDA_Project.git
```

Install dependencies:

```bash

```

### Run the Project

Open Jupyter Notebook:

```bash
jupyter notebook
```

Open:

```text
EDA_Project.ipynb
```

### Results

* TV advertising has the highest impact on sales.
* Radio advertising moderately influences sales.
* Newspaper advertising has minimal influence.
* Correlation analysis helps identify key drivers of sales.
